# Suggested Commands

## Development
- `npm run dev` — Start Astro dev server with HMR
- `npm run build` — Production build (output to `./dist`)
- `npm run preview` — Preview production build locally
- `npm run check` — Run `astro check` (TypeScript + Astro diagnostics)

## Deployment
- Deployed to Cloudflare Pages via Wrangler
- `wrangler.jsonc` configures static asset serving from `./dist`

## System Utilities
- `git` — Version control
- `ls`, `cd`, `grep`, `find` — Standard Linux file navigation
- `node`, `npm` — Node.js runtime and package manager (>=22.12.0)
